# 🚀 Featured Projects

## 🩺 AI-Driven Sleep Apnea Severity Prediction

- Built a CNN–Transformer framework for ECG-based sleep apnea severity prediction.
- Combined **Apnea-ECG, UCDDB and MIT-BIH SLPDB** datasets (76 patients).
- Implemented HRV extraction, ECG morphology analysis, domain adaptation and patient-level evaluation.
- Achieved **QWK 0.575**, **ROC-AUC 0.738**, and **Binary ROC-AUC 0.833**.
